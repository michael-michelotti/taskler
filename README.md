# Tasklr
Task and schedule tracking app, a mix between `Google Keep` and `Jira`

### Choose view for your project
You can display the tasks in your project in a kanban, calendar, or checklist format!

> "Always keep track of your tasks!"

### Create Template Tasks at Set Intervals
Recurring tasks? Set them up as recurring tasks!

### Task Notifications
Get reminded of new tasks and task due dates via email or text!
